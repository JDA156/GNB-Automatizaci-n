# GNB-Automatizaci-n
La automatizacion fue realizada con Jupyter Notebook, con Python3 y Selenium

1. Instalar Python3 sino se tiene instalado desde la URL "https://www.python.org/downloads/" descargar la ultima version e instalar desde el link que aparece debajo del boton grande que muestra Download Python Install Manager NO desde ahi sino el LINK debajo que trae el nombre de la version (recordar seleccionar el check para instalar tambien el path para que se agreguen las variables de entorno).
2. Ingresar al cmd desde el buscador de Windows o con las teclas Windows+R e ingresar CMD, desde MAC ingresar a la terminal
Ingresar los siguiente comandos para verificar instalacion de python (cada codigo esta entre <>): <python --version> <python3 --version> <python -m pip --version> <python3 -m pip --version>
3. Instalar Notebook de Jupyter con el siguiente comando: pip install notebook
Si como en mi caso la version del instalador no deja ejecutar correctamente la instalacion ejecuta el siguiente comando para poder continuar con las instalacion de Notebook: python.exe -m pip install --upgrade pip
Ahora si ejecutamos el comando anterior: pip install notebook
4. Para abrir el Notebook de Jupiter ejecutan el siguiente comando cada que quieran abrir el libro: jupyter notebook
5. Al abrir el Notebook de Jupyter nos vamos a la parte izquierda donde dice desplegamos la lista y seleccionamos Python3
Nos lleva a las celdas donde podemos escribir codigo de python pero en esta ocacion necesitamos abrir el script creado por tal razon nos vamos a el menu y seleccionamos luego y buscamos nuestro Script
6. Finalizamos con abrir el .ipynb y luego en el menu esta vez seleccionamos y y con esto deberia ejecutarse secuencuialmente todas las celdas y abrir nuestra automatizacion correctamente
